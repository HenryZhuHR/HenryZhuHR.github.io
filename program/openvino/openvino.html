# 介绍

介绍大多来自于官方翻译

OpenVINO™ 工具套件 是一个用于深度学习模型优化和部署推理的开源工具包 (openvinotoolkit/openvino)，查看最新文档

开源版本包括组件：模型优化器 Model Optimizer、OpenVINO™运行库 OpenVINO™ Runtime、训练后优化工具 Post-Training Optimization Tool，以及 CPU、GPU、GNA、多设备和异构插件用于在 Intel® CPUs 和 Intel® Processor Graphics 上进行加速

OpenVINO™ 运行库支持多种硬件设备，包括:

  • 「Intel CPU」
  • 「ARM CPU」: Raspberry Pi™ 4 Model B, Apple® Mac mini with M1 chip, NVIDIA® Jetson Nano™, Android™ devices
  • 「Intel GPU」
  • 「Intel GNA」

讲道理，支持 ARM GPU 就有点绷不住了，看到支持 Apple Silicon 就彻底绷不住了

# 模型准备

部署深度学习模型的经典工作流如图

其中 OpenVINO 中间表示(Intermediate Representation, IR) 是一对模型的描述文件:

  • .xml - 描述网络结构 (topology)。
  • .bin - 包含二进制的模型权重。

Model Optimizer mo 将支持的模型转换为 OpenVINO IR，以便在 OpenVINO 推理引擎中使用，支持的模型:

  • TensorFlow
  • TensorFlow Lite
  • PaddlePaddle
  • ONNX

# 模型转换

对于 Pytorch 模型，使用 torch.onnx.export 将模型转换为 ONNX 格式

使用 mo 将 ONNX 模型转换为 OpenVINO IR， mo 有命令行和pyhon 的方式，推荐 python ，参考 Specifying input_shape parameterle，对于支持的层参考 Supported ONNX Layers

打印帮助

from openvino.tools.mo import convert_model
ov_model = convert_model(help=True)

转换模型

from openvino.tools.mo import convert_model
INPUT_MODEL="model.onnx"
ov_model = convert_model(
  INPUT_MODEL, 
  input_shape=[2,300,300,3]
)

# 模型推理

OpenVINO™ 运行时是一组包含 C 和 Python 绑定的 C++ 库，提供通用 API，以在您选择的平台上提供推理解决方案。

使用 OpenVINO 运行时 API 读取中间表示 (IR)、TensorFlow、TensorFlow Lite、ONNX 或 PaddlePaddle 模型，并在首选设备上执行。

OpenVINO™ 运行时使用插件架构。它的插件是软件组件，可完全实现对特定英特尔® 硬件设备（CPU、GPU、GNA 等）进行推理。每个插件都实现统一的 API，并提供额外的硬件特定 API，用于配置设备或在 OpenVINO™ 运行时和底层插件后端之间实现 API 互操作性。
