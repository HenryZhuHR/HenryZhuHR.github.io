


# 通信系统
## 通信系统的主要性能指标
- **有效性** : 消息传输的速度问题。传输一定信息量时，所专用的信道资源/频带宽度/时间间隔
  - 模拟通信系统的有效性 : 被传输信号的有效带宽，即能否充分利用信道资源。在传输信息量不变情况下，系统所占有的有效贷款越小，有效性越好
  - 数字系统 : 码元传输速率（波特率，单位 $Buad=Symbol/s$） $R_B$ 、信息传输速率 $R_b$
- **可靠性** : 消息传输的质量问题。接受信息的准确性
  - 模拟 : 解调器输出信噪比、调制制度增益 $G$ 。
  - 数字 : 误码率 $P_e$ ，误信率

有效性和可靠性是一对矛盾

对于模拟通信系统来说，增加发送信号功率以提高信噪比，可以提高可靠性

> [!NOTE|label:概念]
> **信噪比** (Signal to Noise Ratio, SNR)，信号与噪声之比。信号 $10\log\frac{S}{N}$，功率$20\log\frac{S}{N}$

#### 极化复用 PDM

#### 波分复用 WSM
波分复用 (wavelength-division multiplexing,WSM) 针对光纤通信

是将一系列载有信息、但波长不同的光信号合成一束，沿着单根光纤传输；在接收端再用某种方法，将各个不同波长的光信号分开的通信技术。 这种技术可以同时在一根光纤上传输多路信号，每一路信号都由某种特定波长的光来传送，这就是一个波长信道。 在同一根光纤中同时让两个或两个以上的光波长信号通过不同光信道各自传输信息，称为光波分复用技术，简称 WDM 。 光 波分复用 包括 频分复用 和波分复用。 光频 分复用（frequency-division multiplexing,FDM）技术和 光波分复用 （WDM）技术无明显区别，因为光波是电磁波的一部分，光的频率与波长具有单一对应关系。
