# CARLA 的安装

CARLA 是一个开源的自动驾驶(autonomous driving)模拟器

虚幻引擎 下载 https://www.unrealengine.com/zh-CN/download

CARLA 的安装有 遵照官方编译Linux build:

  1. 🛠 Unreal Engine 安装： 从 0.9.12 版本开始，加入 Unreal Engine 编译
  2. 🛠 carla 编译

切换为 dev 分支，大部分问题都可以被解决

# Unreal Engine 安装

虚幻引擎 (Unreal Engine) 的 源码 是 Epic Games 的私有仓库，在 Accessing Unreal Engine source code on GitHub 页面按照流程注册并申请权限，才能从 Github 上下载源码

参照 Unreal Engine:

  1. clone UnrealEngine 源码，同时选择分支 carla
git clone --depth 1 -b carla https://github.com/CarlaUnreal/UnrealEngine.git
  1. 进入目录
cd UnrealEngine
  1. 进入目录，编译
./Setup.sh \
&& ./GenerateProjectFiles.sh \
&& make

./Setup.sh 可以在本机提前执行，然后推送至无法科学上网的设备，提前执行后，打包 tar -cjvf UnrealEngine.setup.tar.xz UnrealEngine.setup

  1. 启动虚幻引擎编辑器验证软件是否被正确安装
cd ./Binaries/Linux
./UE4Editor

# carla 编译(源码包)

git clone https://github.com/carla-simulator/carla
cd carla

# 编译 Ninja

export INSTALL_PREFIX=~/program/ninja
export NUM_CORES=`nproc --all`         # for Linux

cd <ninja-src>
mkdir -p build
cd build
cmake .. -DCMAKE_INSTALL_PREFIX=$INSTALL_PREFIX
make -j${NUM_CORES}
make install

将下面内容写入环境变量 .bashrc

export NINJA_HOME="$PROGRAM_DIR/ninja"         # ninja
export PATH=$PATH:$NINJA_HOME/bin

# 编译 LLVM

export INSTALL_PREFIX=~/program/llvm
export NUM_CORES=`nproc --all`         # for Linux

cd <ninja-src>
mkdir -p build
cd build
cmake ../llvm \
    -G "Unix Makefiles" \
    -DCMAKE_INSTALL_PREFIX=$INSTALL_PREFIX \
    -DCMAKE_BUILD_TYPE=RelWithDebInfo \
    -DLLVM_ENABLE_PROJECTS="clang;libcxx;libcxxabi" \
    -DBUILD_SHARED_LIBS=ON \
    -DPYTHON_EXECUTABLE=$(which python3)
make -j${NUM_CORES}
make install

将下面内容写入环境变量 .bashrc

export LLVM_HOME="$HOME/program/llvm"
export PATH=$LLVM_HOME/bin:$PATH
export CC=$LLVM_HOME/bin/clang
export CXX=$LLVM_HOME/bin/clang++

# 编译 BOOST

export INSTALL_PREFIX=~/program/boost
export NUM_CORES=`nproc --all`         # for Linux

export INSTALL_PREFIX=$PROGRAM_DIR/$LIBRARY_NAME-$VERSION
py3=`which python3`
py3_root=`${py3} -c "import sys; print(sys.prefix)"`

# configure
./bootstrap.sh \
    --with-python=${py3} --with-python-root=${py3_root} \
    --with-libraries=python,filesystem,system,program_options

./b2 stage release \
    --prefix=$INSTALL_PREFIX \
    --build-dir=./build
./b2 install \
    --prefix=$INSTALL_PREFIX \
    --build-dir=./build

export BOOST_DIR="$PROGRAM_DIR/boost"    # boost

# 编译 CARLA

carla 依赖于 clang ， 但是 carla 默认编译采用 clang-8 ，并且在找不到/usr/bin/clang++-8 则会报错，因此需要修改 calra 默认脚本 <path-to-carla>/Util/BuildTools/Setup.sh

在 Set up environment 内容下，需要修改如下：

  1. 注释 command -v 对于/usr/bin/clang++-8 的检查
  2. 删除CXX_TAG变量，
# ==============================================================================
# -- Set up environment --------------------------------------------------------
# ==============================================================================
# command -v /usr/bin/clang++-8 >/dev/null 2>&1 || {
#   echo >&2 "clang 8 is required, but it's not installed.";
#   exit 1;
# }
# CXX_TAG=c8
export CC=$LLVM_HOME/bin/clang
export CXX=$LLVM_HOME/bin/clang++

在 LLVM 编译项目下面仅保留如下声明

# ==============================================================================
# -- Get and compile libc++ ----------------------------------------------------
# ==============================================================================
LLVM_INCLUDE=${LLVM_HOME}/include/c++/v1
LLVM_LIBPATH=${LLVM_HOME}/lib

在 BOOST 编译项目下面仅保留如下声明

# ==============================================================================
# -- Get boost includes --------------------------------------------------------
# ==============================================================================
BOOST_INCLUDE=${BOOST_DIR}/include
BOOST_LIBPATH=${BOOST_DIR}/lib

RPCLIB_LIBCXX_INCLUDE=${PWD}/${RPCLIB_BASENAME}-libcxx-install/include
RPCLIB_LIBCXX_LIBPATH=${PWD}/${RPCLIB_BASENAME}-libcxx-install/lib
RPCLIB_LIBSTDCXX_INCLUDE=${PWD}/${RPCLIB_BASENAME}-libstdcxx-install/include
RPCLIB_LIBSTDCXX_LIBPATH=${PWD}/${RPCLIB_BASENAME}-libstdcxx-install/lib

XERCESC_VERSION=3.2.3 失效了，需要修改为 XERCESC_VERSION=3.2.4

# 安装 carla

安装完成后，在目录 <path_to_carla_dir>/PythonAPI/carla/dist 下可以找到 carla-0.9.13-cp38-cp38-linux_x86_64.whl 文件

import glob
import os
import sys

try:
    sys.path.append(glob.glob('../carla/dist/carla-*%d.%d-%s.egg' % (
        sys.version_info.major,
        sys.version_info.minor,
        'win-amd64' if os.name == 'nt' else 'linux-x86_64'))[0])
except IndexError:
    pass
