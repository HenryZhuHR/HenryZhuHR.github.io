# 分类问题的损失函数

分类问题中通常采用交叉熵损失函数，在二分类问题中，BCE+Sigmoid ，多分类问题中，CE+Softmax

# BCE

BCE (Binary Cross Entropy) 一般用于二分类任务，是计算模型预测输出 y^​=f(x) 和真实标签 y 之间的误差

LBCE​(y,y^​)=−N1​i=1∑N​[yi​⋅logy^​i​+(1−yi​)⋅log(1−y^​i​)]

# BCE 与 Sigmoid 函数

网络进行反向传播的时候，BCE 求导

∂xi​∂LBCE​​​=∂y^​i​∂LBCE​​⋅∂xi​∂y^​i​​=−[y^​i​yi​​−1−y^​i​1−yi​​]∂xi​∂y^​i​​​

当网络的激活函数选择 Sigmoid 函数 S(x)=1+e−x1​ 时，根据 Sigmoid 函数求导的性质 ∂xi​∂y^​i​​=S(xi​)(1−S(xi​)) ，则

∂xi​∂LBCE​​​=∂y^​i​∂LBCE​​⋅∂xi​∂y^​i​​=−[S(xi​)yi​​−1−S(xi​)1−yi​​]S(xi​)(1−S(xi​))=S(xi​)−yi​​

所以当网络采样 Sigmoid 函数和 BCE 损失函数进行训练时，网络反向传播的梯度与误差是正相关的。所以在二分类问题中，常采用 Sigmoid 函数作为损失函数。
