# Point Cloud Generation

点云生成的方法把点云看成 N×3 的矩阵，点云生成问题转化成矩阵生成问题，于是就有许多生成方法（VAE GAN）

在这项工作中，我们专注于点云的生成，这是一个条件性的生成问题，因为我们工作中考虑的马尔可夫链生成点云的点是以一些形状潜势为条件的。 In this work, we focus on point cloud generation, which is a conditional generation problem, because the Markov chain considered in our work generates points of a point cloud conditioned on some shape latent.
